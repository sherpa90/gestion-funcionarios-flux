Formatos Soportados

Para archivos Excel (.xlsx/.xls): El archivo debe contener las siguientes columnas:

  • Columna A - R.U.T.: Número de RUT (con puntos y guiones)
  • Columna B - Nombre: Nombre completo del funcionario
  • Columna C - Horario: Fecha y hora en formato "DD-MM-YYYY HH:MM"

Para archivos PDF con formato "RUT, Nombre Horario":

  • Ejemplo: 12345678-9, Juan Pérez 08:30-17:30
  • La fecha se toma de los campos Mes/Año que especificas abajo
  • El horario puede ser "08:30" o "08:30-17:30" (hora salida opcional)

💡 Ventajas del formato AM/PM:

  • • Registra hora de entrada (AM) y hora de salida (PM) por separado
  • • Calcula automáticamente el tiempo trabajado
  • • Más preciso para control de asistencia
  • • Compatible con relojes que registran entrada/salida

Notas Generales

  • • Los RUTs pueden tener puntos y guiones (se normalizan automáticamente)
  • • El sistema detecta automáticamente el formato del archivo
  • • La primera fila se considera encabezados y se ignora
  • • Los registros se comparan automáticamente con los horarios estipulados
  • • Se calculan retrasos y tiempos trabajados cuando hay hora de salida
  • • Los registros existentes se actualizan (no se duplican)
